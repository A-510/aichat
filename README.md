## 📌 项目简介

本项目是一个**全栈多模态智能问答系统**，由两大核心模块组成：

| 模块 | 说明 |
|------|------|
| **后端智能体** | 基于腾讯元器平台的复杂工作流编排，实现意图识别、知识检索、多轮问答、文件处理等 |
| **前端界面** | 精美的 Web 聊天界面，支持文本输入、文件上传、Markdown 渲染、打字机效果等 |

### 适用场景

- 📚 知识库问答与检索
- 🖼️ 图片内容识别与分析
- 📄 文档上传与智能解析
- 🎬 视频/音频链接内容提取
- 🌐 联网搜索增强回答
- 😊 趣味表情包检索

## ✨ 功能特性

### 🧠 智能体能力

| 功能 | 描述 | 实现方式 |
|------|------|---------|
| 🎯 意图识别 | 自动识别用户输入类型（文本/图片/文件/链接） | 意图分类节点 |
| 📖 知识库检索 | 基于向量检索，从知识库中查找相关内容 | 大模型知识库查找节点 |
| 🔄 循环问答 | 多轮迭代深入问答，自动判断退出 | 循环节点 + exit_flag |
| 🖼️ 图片处理 | 上传图片后自动识别内容并回答 | 参数提取 + 知识提取 |
| 📄 文件解析 | 支持上传文档文件，智能解析内容 | 上传文件解析插件 |
| 🌐 联网搜索 | 知识库无法回答时自动触发网络搜索 | 联网搜索插件 |
| 😊 表情包检索 | 根据语境推荐相关表情包 | 表情包检索插件 |
| 🎬 多媒体处理 | 支持视频/音频链接的内容提取 | 专用处理分支 |
| 💾 上下文记忆 | 支持多轮对话记忆，保持上下文连贯 | 变量传递 + 赋值节点 |

### 🎨 前端特性

| 功能 | 描述 |
|------|------|
| 💬 实时对话 | 流式输出 + 打字机效果 |
| 📝 Markdown 渲染 | 支持代码高亮、表格、列表等 |
| 📎 文件上传 | 拖拽/点击上传图片和文档 |
| 🌙 深色模式 | 支持亮色/深色主题切换 |
| 📱 响应式布局 | 完美适配桌面端和移动端 |
| ⌨️ 快捷键支持 | Enter 发送，Shift+Enter 换行 |
| 🔄 会话管理 | 新建/清空/导出对话记录 |

## 🛠️ 技术栈

### 后端（智能体工作流）

| 类别 | 技术/工具 | 版本/规格 | 说明 |
|------|-----------|----------|------|
| **开发平台** | 腾讯元器（Yuanqi） | - | 智能体开发与部署 |
| **工作流引擎** | 元器工作流编排 | - | 可视化拖拽式设计 |
| **主推理模型** | 混元 hunyuan-pro | 32K 上下文 | 复杂推理与内容生成 |
| **辅助模型** | hunyuan-lite | - | 轻量判断与分类任务 |
| **阅读理解** | youtu-mrc-pro | 16K 上下文 | 文档阅读理解 |
| **知识库** | 元器知识库 | 向量检索 | 领域知识存储与检索 |
| **插件-联网搜索** | 元器插件市场 | - | 实时网络信息获取 |
| **插件-表情包** | 元器插件市场 | - | 表情包语义检索 |
| **插件-文件解析** | 元器插件市场 | - | 文档内容提取 |

### 前端（Web 聊天界面）

| 类别 | 技术 | 说明 |
|------|------|------|
| **结构** | HTML5 | 语义化标签 |
| **样式** | CSS3 | Flexbox 布局、动画、响应式设计 |
| **交互** | JavaScript (ES6+) | 原生 JS，无框架依赖 |
| **Markdown** | Marked.js | Markdown 解析渲染 |
| **代码高亮** | Highlight.js | 多语言代码高亮 |
| **数学公式** | KaTeX | LaTeX 公式渲染 |
| **接口通信** | Fetch API / SSE | 流式数据传输 |
| **字体图标** | Font Awesome | UI 图标库 |
